# TLoL Agent

## Overview

Using [various](https://huggingface.co/datasets/maknee/league-of-legends-decoded-replay-packets/tree/main)
[datasets](https://huggingface.co/datasets/maknee/league-of-legends-decoded-replay-packets-s12-unorganized)
from [maknee](https://github.com/maknee) (rougly 140,000 replays) to build a Human level agent for League of Legends.

Also using the [accompanying](https://github.com/Maknee/league-of-legends-decoded-replay-packets-gym/) library to
process the files.
